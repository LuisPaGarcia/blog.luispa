## tl;dr
- Puppeteer es una biblioteca usada con Node.js que proporciona una API de alto nivel para controlar una instancia de Chrome o Chromium headless a través del protocolo de DevTools. 
- La API de Puppeteer permite realizar acciones simples de manera rápida y controlable.
- Usé puppeteer para obtener información de ciertas compras registradas en el portal **Guatecompras** usando un script simple.
